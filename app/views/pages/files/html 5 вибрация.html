Vibration API позволяет вибрировать мобильное устройство с заданной продолжительностью. Определение поддержки Vibration API устройством: 
var supportsVibrate = "vibrate" in navigator;
Базовое использование: Метод navigator.vibrate принимает один аргумент, число для одиночной вибрации или массив чисел для серии вибраций. 
//Вибрировать один раз в течении секунды
navigator.vibrate(1000);

//Вибрировать три раза с продолжительностью 2, 3 и 1 секунда
navigator.vibrate([2000, 3000, 1000])

//Чтобы остановить вибрацию нужно передать в качестве аргумента ноль или пустой массив
navigator.vibrate(0);
navigator.vibrate([]);
